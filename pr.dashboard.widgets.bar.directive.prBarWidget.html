prBarWidget
directive in module pr.dashboard.widgets.bar

Description

The prBarWidget pulls data and presents it as a bar chart based on the parameters.

Usage

as element:
<pr-bar-widget
       params="{object}"
       [options="{object}"]
       filters="{object}"
       [transform-data="{function}"]>
</pr-bar-widget>
