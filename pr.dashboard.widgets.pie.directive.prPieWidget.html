prPieWidget
directive in module pr.dashboard.widgets.pie

Description

The prPieWidget pulls data and presents it as a pie chart based on the parameters.

Usage

as element:
<pr-pie-widget
       params="{object}"
       [options="{object}"]
       filters="{object}"
       [transform-data="{function}"]>
</pr-pie-widget>
