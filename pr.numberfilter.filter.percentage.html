percentage
filter in module pr.numberfilter

Description

Formats a number as a percentage. (e.g. 0.45 -> 45%)

Usage

In HTML Template Binding

{{ percentage_expression | percentage:fractionSize }}

In JavaScript

$filter('percentage')(n, fractionSize)

Parameters

ParamTypeDetails
nnumber

Amount to format using 1 as 100%.

fractionSizenumber

Number of decimal places to round the number to, defaults to 2.
