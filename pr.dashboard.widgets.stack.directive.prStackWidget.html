prStackWidget
directive in module pr.dashboard.widgets.stack

Description

The prStackWidget pulls data and presents it as a stacked area char based on the parameters.

Usage

as element:
<pr-stack-widget
       params="{object}"
       [options="{object}"]
       filters="{object}"
       [transform-data="{function}"]>
</pr-stack-widget>
