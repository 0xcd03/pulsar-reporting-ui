prCountTo
directive in module pr.countto

Description

The prCountTo directive displays an animated numeric count using jQuery countTo.

https://github.com/mhuggins/jquery-countTo

If the plugin is not present, the contents will be replaced directly in to the html tag.

The animation lasts for 1 second.

This directive makes use of both $timeout and setInterval to animate the number.

Usage

as element:
<pr-count-to
       value="{number}"
       decimals="{number}"
       placeholder="{string}">
</pr-count-to>
as attribute
<ANY pr-count-to
     value="{number}"
     decimals="{number}"
     placeholder="{string}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
valuenumber

The value to animate, animation will happen every time it changes.

decimalsnumber

Number of decimals

placeholderstring

Text to show in case the value is not a number: undefined or null
