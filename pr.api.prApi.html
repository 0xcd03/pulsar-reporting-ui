prApi
service in module pr.api

Description

An object that represents the connection to the Pulsar Query API. Contains the url of the API and other basic configuration.

Properties

  • timezone

    Timezone of the backend

  • url

    Url of the Pulsar Query API backend

  • withCredentialsDatasource

    Whether to set the withCredentials flag on the requests to the datasources
