pr-grid
object in module pr.dashboard.widgets.grid

Description

Widget for ui-grid. This widget displays data in a table format and performs very little formatting of the original data.

It supports both several dimensions and several metrics simultaneoustly which allows users to explore data in detail. ui-grid offers high performance for high quantities of rows of data.

Usage:

HTML:

<pr-dashboard-widget widget="widget"></pr-dashboard-widget>

JS:

$scope.widget = {
  "type": "pr-grid",
  "params": {
    "dimensions": [
      {
        "name": "browserfamily"
      },
      {
        "name": "trafficsource"
      }
    ],
    "metrics": [
      {
        "name": "clickcount_ag",
        "type": "count",
        "alias": "clickcount_ag"
      }, {
        "name": "gmv_ag",
        "type": "sum",
        "alias": "gmv_ag"
      }
    ],
    "maxResults": 100
    "granularity": "all",
    "dataSourceName": "pulsar_ogmb"
  },
  "options": {
    "title": "A grid widget",
    "disabled": false,
    "grid" : {
       // Additional options that will be sent ui-grid, see http://ui-grid.info/docs/#/tutorial
    }
  }
}
