prDatepicker
directive in module pr.date

Description

The prDatepicker directive creates a date picker for a range of dates.

Dependencies

  • bootstrap-daterangepicker

    https://github.com/dangrossman/bootstrap-daterangepicker

  • momentjs

    http://momentjs.com/

Usage

as element:
<pr-datepicker
       start-date="{number}"
       end-date="{number}"
       callback="{function}"
       [format="{string}"]
       [timezone="{string}"]>
</pr-datepicker>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
startDatenumber

The starting date of the range, in timestamp format

endDatenumber

The starting date of the range, in timestamp format

callbackfunction

The callback to run when the dates range is changed

format
(optional)
string

The date format to display, defaults to YYYY-MM-DD

timezone
(optional)
string

The timezone to use when creating the ranges.
