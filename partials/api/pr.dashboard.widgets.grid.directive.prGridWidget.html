prGridWidget
directive in module pr.dashboard.widgets.grid

Description

The prGridWidget pulls data and presents it as a grid based on the parameters.

Usage

as element:
<pr-grid-widget
       params="{object}"
       [options="{object}"]
       filters="{object}"
       [transform-data="{function}"]>
</pr-grid-widget>
