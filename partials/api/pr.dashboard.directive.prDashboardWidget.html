prDashboardWidget
directive in module pr.dashboard

Description

The prDashboardWidget creates a single widget.

A widget is a representation of data to the user. It is typically shown as a chart or table to display data in a visually appealing way.

The widget directive delegates its body of content to one of the widget directives (bar, grid, linewithfocus, ...) according to the widget type (widget.type).

Usage

as element:
<pr-dashboard-widget
       widget="{object}"
       filter="{object}"
       [edit-mode="{boolean}"]
       [widgets="{object}"]>
</pr-dashboard-widget>

Parameters

ParamTypeDetails
widgetobject

The object that displays the widget

filterobject

The query filter

editMode
(optional)
boolean

Flag to enable the widget to be editable from the UI

widgets
(optional)
object

Reference to array of widgets, so the widget can remove itself from a report.
