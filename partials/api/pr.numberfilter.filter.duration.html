duration
filter in module pr.numberfilter

Description

Formats a number in miliseconds as a duration in the format HH:mm:ss (e.g. 01:59:59)

Usage

In HTML Template Binding

{{ duration_expression | duration }}

In JavaScript

$filter('duration')(n)

Parameters

ParamTypeDetails
nnumber

Amount in miliseconds to format, can be negative.
