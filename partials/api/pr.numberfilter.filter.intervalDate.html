intervalDate
filter in module pr.numberfilter

Description

Formats time to be used in the queries

Usage

In HTML Template Binding

{{ intervalDate_expression | intervalDate[:format][:originalFormat][:timezone] }}

In JavaScript

$filter('intervalDate')(time[, format][, originalFormat][, timezone])

Parameters

ParamTypeDetails
timestringnumber

Timestamp to be converted

format
(optional)
string

Format to use in returning the date. Defaults to 'X' (timestamp)

originalFormat
(optional)
string

Format. Defaults to 'YYYY-MM-DD HH:mm:ss'

timezone
(optional)
string

Timezone
