prMetricWidget
directive in module pr.dashboard.widgets.metric

Description

The prMetricWidget pulls data and presents it as an animated metric.

Usage

as element:
<pr-metric-widget
       params="{object}"
       [options="{object}"]
       filters="{object}"
       [transform-data="{function}"]>
</pr-metric-widget>
