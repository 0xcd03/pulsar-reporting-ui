prTimelineWidget
directive in module pr.dashboard.widgets.timeline

Description

The prTimelineWidget pulls data and presents it as a time line chart based on the parameters.

Usage

as element:
<pr-timeline-widget
       params="{object}"
       [options="{object}"]
       filters="{object}"
       [transform-data="{function}"]>
</pr-timeline-widget>
