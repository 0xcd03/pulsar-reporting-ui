prApiProvider
service in module pr.api

Description

The prApiProvider is used to configure the connection to the Pulsar Query API.

Methods

  • timezone(timezone)

    Configures the timezone which will be used in generating dates, should match the back-end timezone.

    Parameters
    ParamTypeDetails
    timezonestring

    Must match a timezone provided by moment timezone. Defaults to 'MST'

  • url(url)

    Configures the URL which will be used to call the backend (Pulsar Query API).

    Parameters
    ParamTypeDetails
    urlstring

    Base URL for queries to the back-end dataSource. Defaults to the empty string'', in which case the back-end calls will be done to the same URL as the application in the browser. Don't add a trailiing slash.

  • useWithCredentialsDatasource(value)

    Configures the withCredentials flag in all XHR requests to the datasources. If not configured, defaults to 'false'

    Parameters
    ParamTypeDetails
    valueboolean

    Values of the withCredentials flag in all requests to the datasource. Defaults to 'false'
