prLineWithFocusWidget
directive in module pr.dashboard.widgets.linewithfocus

Description

The prTimelineWidget pulls data and presents it as a line chart with a focus area based on the parameters.

Usage

as element:
<pr-line-with-focus-widget
       params="{object}"
       [options="{object}"]
       filters="{object}"
       [transform-data="{function}"]>
</pr-line-with-focus-widget>
