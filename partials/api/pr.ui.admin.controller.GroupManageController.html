GroupManageController
controller in module pr.ui.admin

Description

The GroupManageController is used to control admin page group management table. For Wire all features workable, only need to provide below dependencies which will be injected into this controller. Specifically to say, users only need to provide valid api service as clarified in related API Call Service, and load related module, then admin module's features can be set up rightly.

Dependencies

  • GroupSearchService
  • prUIOptionService
  • $modal

    https://github.com/angular-ui/bootstrap/tree/master/src/modal/docs

  • Groups

    API service used to interact with backend to get the user's groups

  • GroupUsers

    API service used to interact with backend to get the group's users

  • GroupRights

    API service used to interact with backend to get the group's rights

  • Datasources

    API service used to interact with backend to get the user's rights on datasource

  • SysRights

    API service used to interact with backend to get the user's system rights

  • prDashboardResource

    API service used to interact with backend to get the user's rights on prDashboardResource
